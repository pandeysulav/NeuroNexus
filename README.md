```bash

cd daily-learning-progress
```

### 2. **Create a New Day Entry**
```bash
mkdir daily_logs/day_XXX
cd daily_logs/day_XXX

```

### 3. **Add Your Daily Learning**
- Write notes in `notes.md`
- Code in `day_XXX.ipynb` (Jupyter notebook)
